# ProxyChecker
MultiThreading Proxy cheker

# Installing
```
git clone https://github.com/freQuensy23-coder/ProxyChecker/
pip install requests
```
# Using:
```
python CommandLineUtil.py --file proxy.txt --threads -1 --timeout 20
```
Result will be saved to Goods.txt
